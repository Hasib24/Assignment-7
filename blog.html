1. What are the difference between map(), filter() and forEach() in Javascript ?

map( )

map( ) creates a new array for every array element. map( ) calls a function once for each element in an array. map( ) does not execute the function for empty elements. map( ) does not change the original array.

filter( )

filter() makes a new array of elements that pass condition. It does not execute for empty elements. filter() method not change the original array.

forEach( )

forEach() method execute for every array elements but it returns nothing or undefined.It aslo does not modify the original array.

find( )

The find() method returns the value of the first element that mached the condition. The find() method executes a function for each array element. It returns undefined if no elements are matched.

3. What is event loop ?
Event loop is the system of managing the order in which tasks are executed in the code. It is a part of asynchronous programming model. It continuously monitors the execution stack and the task queue. When the execution stack is empty, the event loop looks at the task queue and picks the next task to be executed.
